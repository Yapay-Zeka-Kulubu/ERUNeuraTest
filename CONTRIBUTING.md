# Katkıda Bulunma Rehberi

ERUNeuraTest'e katkıda bulunmak istediğiniz için teşekkür ederiz! 🎉

## 🚀 Nasıl Katkıda Bulunabilirim?

### Hata Bildirimi
1. [Issues](https://github.com/Yapay-Zeka-Kulubu/ERUNeuraTest/issues) sayfasını kontrol edin
2. Yeni bir issue açın ve hatanın detaylarını paylaşın
3. Mümkünse hata mesajını ve kullandığınız ortamı belirtin

### Yeni Özellik Önerisi
1. Önce bir issue açarak önerinizi tartışın
2. Onay aldıktan sonra PR oluşturun

### Pull Request Süreci
1. Repo'yu fork edin
2. Yeni bir branch oluşturun (`git checkout -b feature/amazing-feature`)
3. Değişikliklerinizi commit edin (`git commit -m 'feat: Add amazing feature'`)
4. Branch'inizi push edin (`git push origin feature/amazing-feature`)
5. Pull Request açın

## 📝 Kod Standartları

- Python kodu için [PEP 8](https://pep8.org/) stilini takip edin
- Fonksiyonlar için docstring yazın
- Test eklemeyi unutmayın

## 🧪 Test Çalıştırma

```bash
.venv\Scripts\activate
python -m pytest
```

## 📋 Commit Mesajları

[Conventional Commits](https://www.conventionalcommits.org/) formatını kullanın:

- `feat:` Yeni özellik
- `fix:` Hata düzeltmesi
- `docs:` Dokümantasyon
- `refactor:` Kod refactoring
- `test:` Test ekleme

## ❓ Sorular

Sorularınız için [Discussions](https://github.com/Yapay-Zeka-Kulubu/ERUNeuraTest/discussions) kullanabilirsiniz.

Teşekkürler! 🙏
